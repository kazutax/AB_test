# AB_test

A/Bテストの実践に向け、関連して学んだ知見やコードスニペットを置いとくリポジトリ

## 目的

Bayesian A/B Testing について学ぶ

> 従来の頻度論ABテストから脱却するため取り入れた知識をメモしておく。<br>
> p-value や結果の解釈が一人歩きして辛くなるため、ベイズ推論を使った解釈の導入を進める。
